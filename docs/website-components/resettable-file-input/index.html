Resettable file input

To use the resettable file input, add resettable attribute to your file input's wrapper.

Full Usage Example:

<form>
      <div data-resettable-file-input>
        <label for="message_attachments">Attachments (PDF/Images)</label>
        <input type="file" multiple="multiple" />
      </div>
    </form>
