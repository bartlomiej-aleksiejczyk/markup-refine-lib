Simple Topbar Component

For simple cases, class navbar can be used to style the top bar.

To improve apperance of topbar element classes: navbar-brand and group can be used. 


    <header class="navbar">
        <div class="navbar-brand">MySite</div>
        <div class="group">
          <a href="">Home</a>
          <a href="">About Us</a>
          <a href="">Contact</a>
        </div>
    </header>
