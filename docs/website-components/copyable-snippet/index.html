Copyable Snippet

Use the data-copyable-snippet attribute on any <pre> element that wraps a <code> block to enable automatic copy functionality. A copy button will be injected automatically if not already present.

Example usage:

<pre data-copyable-snippet>
  <code>
    npm install my-package
  </code>
</pre>

Rendered snippet:

npm install my-package

When the copy button is clicked, the visible text inside the <code> block (excluding the button itself) will be copied to the clipboard. If the Clipboard API is not available (e.g. in insecure contexts), a fallback method will be used.
