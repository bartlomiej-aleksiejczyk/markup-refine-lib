Navigation Tabs

Use the navigation-tabs class on any element to make it navigation tabs.

Add data-navigation-tabs-autoselector attribute to select currently choose tab.

Example usage:

<nav class="navigation-tabs" data-navigation-tabs-autoselector>
  <a> List Templates</a>
  <a href="/markup-refine-lib/website-components/navigation-tabs"> This Page </a>
  <a> List Jobs</a>
  <a> New Job</a>
</nav>

Rendered snippet:

List Templates This Page List Jobs New Job

Also more semantic version is supported

<nav class="navigation-tabs">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Clients</a></li>
    <li><a href="#">Contact Us</a></li>
  </ul>
</nav>

Rendered snippet:

  • Home
  • About
  • Clients
  • Contact Us

You can use an interactive version of this component that is headless and is not tied to any css

Use the data-tabs-group and data-tabs-name and data-tabs-button-name to group and connect tab's content and buttons. 

<nav class="navigation-tabs">
  <ul>
    <button data-tabs-button-name="demo-1" data-tabs-group="demo">Home</button>
    <button data-tabs-button-name="demo-2" data-tabs-group="demo">About</button>
    <button data-tabs-button-name="demo-3" data-tabs-group="demo">Clients</button>
    <button data-tabs-button-name="demo-4" data-tabs-group="demo">Contact Us</button>
  </ul>
  <p>Choosen tab</p>
    <div data-tabs-name="demo-1" data-tabs-group="demo" data-tabs-default>Default</div>
    <div data-tabs-name="demo-2" data-tabs-group="demo">About</div>
    <div data-tabs-name="demo-3" data-tabs-group="demo">Clients</div>
    <div data-tabs-name="demo-4" data-tabs-group="demo">Contact Us</div>
</nav>

Rendered snippet:

Choosen tab

Default
About
Clients
Contact Us
