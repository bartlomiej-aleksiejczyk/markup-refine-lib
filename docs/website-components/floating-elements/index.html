Floating Elements

Use the floating-element class on any element to make it float.

Example usage:

<figure class="floating-element">
  <img src="https://placehold.co/250x200" alt="Placeholder image 1"/>
  <figcaption style="font-size: 0.9em; text-align: center;">This is a caption</figcaption>
</figure>
