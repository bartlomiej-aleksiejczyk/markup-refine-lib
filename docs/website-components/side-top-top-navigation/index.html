Side Top Top Navigation

This component provides fully responsive navigation bars using a Web Component:

Usage

To use this component, add the following code and insert your own HTML into the topbar's slots: 

      <side-top-top-navigation>
  <div slot="primary-topbar">Primary Topbar</div>
  <div slot="secondary-topbar">Secondary Topbar</div>
  <div slot="sidebar">Sidebar content</div>
</side-top-top-navigation>

Info: You need to add your own styles to the topbar content.

The second thing you need is to adjust your website layout by adding the class stt-navigation-main-content: 

<main class="stt-navigation-main-content responsive-container">
...your content
</main>

Info: As you can see, you can combine the layout class with the responsive-container class to improve page responsiveness.

Styling

To ease the styling of topbar content, you can use the following classes on the topbar's section content: stt-primary-sidebar, stt-primary-topbar, stt-secondary-topbar

These classes set the dimensions of each topbar section.

Full usage example:

      <side-top-top-navigation>
      <div slot="primary-topbar">Primary Topbar</div>
      <div slot="secondary-topbar" class="stt-secondary-topbar">
        Secondary Topbar
      </div>
      <div
        slot="sidebar"
        class="stt-primary-sidebar"
        style="display: flex; flex-direction: column; justify-content: center;"
      >
        <h3 style="text-align: center;">Any content</h3>
      </div>
    </side-top-top-navigation>
    <main class="stt-navigation-main-content responsive-container">
    </main>
